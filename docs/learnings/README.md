# Learning Notes

This folder contains learning notes from contributors. Each note documents what someone learned while working on a task.

## Purpose

- Share knowledge with other contributors
- Create a learning resource for newcomers
- Reflect on and solidify your own learning
- Build a community knowledge base

## How to Add Your Note

1. Copy `TEMPLATE.md` to a new file
2. Name it: `YYYY-MM-DD_your-name_task-name.md`
3. Fill in the sections
4. Submit with your PR or as a separate contribution

## Example

After completing Issue #7 (YOLO integration), you might create:
`2026-01-29_alice_yolo-integration.md`
